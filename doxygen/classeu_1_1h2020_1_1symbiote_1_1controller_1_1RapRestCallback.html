Detailed Description

A Callback for listening to asynchronous REST replies from the Resource Access Proxy

This class extends the ListenableFutureCallback class and uses the DeferredResult class for replying asynchronously to the HTTP Request which received from the external world, as soon as it gets the response from the Resource Access Proxy.
